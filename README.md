---#  How to Run the Project

## Prerequisites

Make sure you have the following installed on your system:

- Node.js (v18 or later recommended)
- npm (comes with Node.js)

Verify the installation:

```bash
node -v
npm -v
```

---

## Step 1: Clone the Repository

```bash
git clone https://github.com/ShivaRohith619/ADAPTIVE-TRAFFIC-SIGNAL-CONTRO-AGENT-USING-Q-LEARNING.gi
```

Navigate to the project directory:

```bash
cd <repository-name>
```

---

## Step 2: Install Dependencies

```bash
npm install
```

---

## Step 3: Configure Environment Variables

Create a `.env` file in the project root.

Copy the contents from `.env.example`:

```bash
cp .env.example .env
```

Update the required API keys or environment variables.

---

## Step 4: Start the Development Server

```bash
npm run dev
```

The application will start and will usually be available at:

```
http://localhost:5173
```

---

## Step 5: Build for Production

```bash
npm run build
```

---

## Step 6: Preview the Production Build

```bash
npm run preview
```

---

## Troubleshooting

If dependencies are missing:

```bash
npm install
```

If the application doesn't start:

```bash
npm install
npm run dev
```

If environment variables are required, ensure the `.env` file is correctly configured.
